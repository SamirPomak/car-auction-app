## 📋 About The Project

## 🚀 Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Node.js
- npm

### Installation

- Clone the repo or download as ZIP
  ```sh
  git clone https://github.com/SamirPomak/car-auction-app.git
  ```
  Install NPM packages
- app
  ```sh
  cd car-auction-app/
  npm install
  ```
